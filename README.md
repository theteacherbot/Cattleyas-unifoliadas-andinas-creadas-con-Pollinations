# 🌺 Carrusel de Cattleyas - Orquídeas Unifoliadas Andinas

Una aplicación web interactiva para explorar y comparar ilustraciones botánicas generadas por IA con imágenes reales de orquídeas del género *Cattleya*, especies unifoliadas andinas.

## ✨ Características

- **🔍 Comparación IA vs Real**: Toggle interactivo para alternar entre ilustraciones botánicas generadas por IA e imágenes fotográficas reales
- **🧬 Navegación por especies**: Selector de especies con indicadores visuales (C. dowiana y más)
- **🖼️ Carrusel interactivo**: Transiciones suaves entre imágenes con controles de navegación
- **📊 Barra de progreso**: Indicador visual con gradiente de colores que muestra el avance
- **🎞️ Miniaturas**: Navegación rápida mediante thumbnails de todas las especies
- **🎨 Diseño elegante**: Interfaz oscura con paleta de colores inspirada en orquídeas (púrpura, dorado, crema)

## 🛠️ Tecnologías

- **HTML5** semántico
- **CSS3** con variables CSS y diseño responsive
- **JavaScript vanilla** para la lógica del carrusel
- **Google Fonts**: Libre Baskerville y Raleway

## 🎯 Propósito

Esta aplicación sirve como herramienta de **cotejamiento visual** entre representaciones artísticas de IA y documentación fotográfica real de especies botánicas, útil para:
- Estudios de botánica
- Validación de representaciones generativas
- Educación sobre diversidad de orquídeas andinas

## 🚀 Uso

1. Haz clic en las **categorías del navegador** para cambiar entre especies
2. Usa los **toggles** (🎨/📷) para alternar entre ilustración e imagen real
3. Navega con las **flechas** o haz clic en las **miniaturas**
4. Observa la **barra de progreso** para saber tu posición

## 📱 Responsive

Diseño adaptativo que funciona en dispositivos móviles y escritorio.
